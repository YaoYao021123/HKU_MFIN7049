# HKU MFIN7049 Study Review

[中文](#中文) | [English](#english)

## 中文

这是一个用于复习 HKU `MFIN7049 Money, Financial Institutions and Markets` 的静态 HTML 复习站点。内容按主题模块组织，从课程大图景开始，逐步进入利率、估值、风险定价、信息摩擦、融资激励、银行挤兑、金融危机与货币政策。

**仓库说明**

- `index.html`：站点入口，支持中文/English 按钮切换介绍，不同时显示两种语言。
- `module-1-review.html`：模块 1，课程大图景与金融体系基础。
- `module-2-review.html`：模块 2，利率、期限结构与估值引擎。
- `codex.md`：后续模块生成规范、复习节奏、题型蓝图和掌握度追踪。
- `.github/workflows/pages.yml`：GitHub Pages 自动部署配置。

**内容原则**

- 主体复习页使用中文讲解，并保留必要英文术语。
- 每一模块包含：主题鸟瞰、核心概念、必会公式、完整例题、互动题组、即时反馈和下轮连接。
- 核心概念默认按 `定义 / 直觉 / 考试问法 / 易错点` 展开。
- 原始课程 PDF、作业 PDF 和课件不会纳入仓库。

**部署**

推送到 `main` 后，GitHub Actions 会把静态 HTML 部署到 GitHub Pages。部署完成后站点地址通常为：

```text
https://yaoyao021123.github.io/HKU_MFIN7049/
```
## English

This repository contains a static HTML study site for HKU `MFIN7049 Money, Financial Institutions and Markets`. The review is organized by topic modules, starting from the big picture of the financial system and moving into interest rates, valuation, risk pricing, information frictions, financing incentives, bank runs, financial crises, and monetary policy.

**Repository Description**

- `index.html`: site entry page with a Chinese/English language toggle. Only one language is shown at a time.
- `module-1-review.html`: Module 1, big picture of the course and financial-system foundations.
- `module-2-review.html`: Module 2, interest rates, term structure, and valuation engine.
- `codex.md`: generation protocol for future modules, exam blueprint, review rhythm, and mastery tracking.
- `.github/workflows/pages.yml`: GitHub Pages deployment workflow.

**Content Rules**

- Main review pages are written in Chinese while preserving necessary English technical terms.
- Each module includes a topic overview, expanded concepts, formulas, worked examples, interactive questions, feedback, and a next-module bridge.
- Core concepts are expanded by `definition / intuition / exam pattern / common mistake`.
- Raw course PDFs, assignment PDFs, and lecture slides are excluded from the repository.

**Deployment**

After pushing to `main`, GitHub Actions deploys the static HTML files to GitHub Pages. The expected site URL is:

```text
https://yaoyao021123.github.io/HKU_MFIN7049/
```
