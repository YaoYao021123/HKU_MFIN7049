# HKU MFIN7049 复习站

[English README](README.en.md)

这是一个用于复习 HKU `MFIN7049 Money, Financial Institutions and Markets` 的静态 HTML 复习站点。内容按主题模块组织，从课程大图景开始，逐步进入利率、估值、风险定价、信息摩擦、融资激励、银行挤兑、金融危机与货币政策。

## 在线访问

部署完成后的站点入口：

```text
https://yaoyao021123.github.io/HKU_MFIN7049/
```

站点会自动跳转到：

```text
https://yaoyao021123.github.io/HKU_MFIN7049/course/
```

## 仓库结构

- `course/index.html`：课程复习站入口，支持中文/English 按钮切换说明，不同时显示两种语言。
- `course/module-1-review.html`：模块 1，课程大图景与金融体系基础。
- `course/module-2-review.html`：模块 2，利率、期限结构与估值引擎。
- `course/codex.md`：后续模块生成规范、复习节奏、题型蓝图和掌握度追踪。
- `.github/workflows/jekyll-gh-pages.yml`：GitHub Pages 自动部署配置。

## 内容原则

- 主体复习页使用中文讲解，并保留必要英文术语。
- 每一模块包含：主题鸟瞰、核心概念、必会公式、完整例题、互动题组、即时反馈和下轮连接。
- 核心概念默认按 `定义 / 直觉 / 考试问法 / 易错点` 展开。
- 原始课程 PDF、作业 PDF 和课件不纳入仓库。
